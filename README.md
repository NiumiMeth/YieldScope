# Bond Portfolio Analytics

Basic bond portfolio analytics starter. Structure provided:

- `analytics/` — bond math: pricing, yield, duration, convexity, risk
- `portfolio/` — loading, metrics, history
- `database/` — sqlite connection helper
- `visualizations/` — plotly charts and Streamlit dashboard pieces
- `utils/` — helpers and date utilities
- `data/` — sample datasets (includes `sample_portfolio.csv`)

Run the app:

```bash
pip install -r requirements.txt
streamlit run app.py
```

Edit files in the folders to add more features and tests.
